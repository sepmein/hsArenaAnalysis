hsArenaAnalysis
===============

A hearth stone arena winning rate analysis tool built by Spencer.

A certain log may contain such things:
{
    user: userid,
    pro_used: mage,
    pro_banned: [warrior, hunter],
    winning_num: 8,
    losing_num: 3,
    best_cards: [],
    worst_cards: [],
    best_cards_op: [],
    pro_hated_most: mage
}
